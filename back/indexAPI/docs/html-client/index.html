Autenticação (autenticacao)
Autentica o usuario

Object

Example data

Content-Type: application/json
""

Responses

200

Login efetuado com sucesso. Object

400

Email ou Senha não enviados.

401

Senha inválida.

404

Usuario não encontrado.

500

Erro interno. ou clubes.model não cincronizado com auth.model
